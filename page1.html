HTML Concepts

  1. Semantic Tags
    1. Separate the HTML into chunks for organization and easy reading
    2. Gives CSS the ability to edit large portions of code easily
  2. CSS Integration
    1. Can change colors of text or backgrounds
    2. Can be used to control the styles for individual IDs, classes, or tags
  3. Lists
    1. Unordered lists standardly create dots before each entry, unless specified otherwise
    2. Ordered lists standardly create numbers before each entry, unless specified otherwise
